工作室新闻

2077年6月6日。悼红轩工作室被Rockstar Games（R星）以666亿美元成交价收购，收购内容包括悼红轩游戏工作室旗下的所有ip，工作室创始人Jack将继续担任工作室的CEO。

游戏新闻

悼红轩工作室获得刘慈欣科幻小说《三体》、《三体Ⅱ：黑暗森林》、《三体Ⅲ：死神永生》版权，VR游戏《三体》将于2077年2月13日发售。
